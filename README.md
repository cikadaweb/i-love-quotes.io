# i-love-quotes.io

## Бесплатный JSON API для взаимодействия с цитатами из ваших любимых фильмов и сериалов.

![screenshot](previews/main.jpg)

### Technologies
* PHP 8.1 / PDO
* SCSS

## Project setup

```sh
docker compose build
```

### Project start

```sh
docker compose up -d
```

### Project stop

```sh
docker compose down
```
